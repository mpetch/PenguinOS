# Penguin OS

## Requirements

### Windows
- Windows SubSystem (WSL)

## How to run

### Linux

1. Run `bash build_cross_compiler.sh`.
2. Run `build.sh`.

### Windows

1. Open a new Command Prompt window.
2. `cd` to the location of this repository.
3. Enter `wsl`.
4. Run `bash build_cross_compiler.sh`.
5. Run `build.sh`.
